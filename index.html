📊 Calcolo indice Pulsatiliy Index"

Peak Detection
Analysis Results
Metodo 1:Flow Velocity (FV)
nCPP = MAP × (FVdia/FVmean) + 14 → ICP = MAP - nCP
Metodo 2: Pulsatility Index (PI)
PI = (FVsys - FVdia) / FVmean → ICP = 4.47 × PI + 12.68
Descrizione Metodo
Basato sulla pressione di perfusione cerebrale (CPP) e sul rapporto
FVdia/FVmean come indicatore di resistenza vascolare distale.
La costante +14 rappresenta la pressione venosa cerebrale media.
Descrzione Metodo
Correlazione diretta tra pulsatilità del flusso cerebrale e ICP.
I coefficienti derivano da regressione lineare su database
multicentrico con validazione invasiva.

⚙️ Vantaggi e Limiti

Vantaggi FV
Robusto a variazioni emodinamiche sistemiche
Basato su principi fisiopatologici consolidati
Meno sensibile a artefatti del segnale
Limiti FV
Richiede calibrazione per diverse popolazioni
Influenzato da compliance vascolare
Vantaggi PI
Correlazione diretta e lineare con ICP
Sensibile a variazioni acute
Ampiamente validato in letteratura
Limiti PI
Influenzato da pressione arteriosa sistemica
Sensibile a Complinace Arteriosa

🎯Prinicipi Fisiopatologici

Metodo FV
ICP normale → Flusso diastolico buono → FVdia/FVmean alto → nCPP alta → ICP calcolata bassa ✓
ICP elevata → Flusso diastolico ridotto → FVdia/FVmean basso → nCPP bassa → ICP calcolata alta ✓
Metodo PI
ICP normale → Vasi rilassati → Flusso diastolico buono → PI basso → ICP calcolata normale ✓
ICP elevata → Vasi compressi → Flusso diastolico ridotto → PI alto → ICP calcolata alta ✓
